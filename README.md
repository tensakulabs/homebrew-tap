# homebrew-tap

Homebrew tap for [Tensaku Labs](https://tensakulabs.com) tools.

## Usage

```bash
brew tap tensakulabs/tap
```

## Formulae

| Formula | Description |
|---------|-------------|
| `tcr` | Token Context Reducer — minimize LLM token consumption in CLI workflows |

### Install tcr

```bash
brew install tensakulabs/tap/tcr
```
